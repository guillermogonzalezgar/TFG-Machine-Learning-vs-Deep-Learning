# TFG: Comparación entre Machine Learning Clásico y Deep Learning

Repositorio que contiene el código desarrollado para el Trabajo Fin de Grado centrado en la comparación de algoritmos clásicos de Machine Learning y redes neuronales multicapa en distintos problemas de clasificación.

## Datasets utilizados

- Adult Income
- Breast Cancer Wisconsin
- MNIST
- Credit Default
- Dataset Simulado No Lineal
- Dataset Simulado con Ruido

## Tecnologías utilizadas

- Python
- Scikit-learn
- XGBoost
- TensorFlow
- Keras
- NumPy
- Pandas
